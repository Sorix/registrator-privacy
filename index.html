Политика конфиденциальности

Приложение: Registrator
Дата обновления: июнь 2026

Registrator — приложение для внутреннего учета и координации рабочих данных на устройстве пользователя.

Какие данные собирает разработчик

Разработчик приложения не собирает, не получает и не хранит персональные данные пользователей. Приложение не отправляет данные на серверы разработчика.

Данные внутри приложения

Пользователь может самостоятельно вводить в приложение сведения о командах, участниках и другую рабочую информацию. Эти данные хранятся локально на устройстве пользователя и не передаются разработчику.

Передача данных пользователем

Пользователь может самостоятельно экспортировать или передавать данные через функции устройства и выбранные им приложения. Такая передача выполняется только по действию пользователя. Разработчик не контролирует и не получает данные, которые пользователь передает таким способом.

Камера и QR-коды

Если приложение запрашивает доступ к камере, он используется только для сканирования QR-кодов внутри приложения. Фото и видео с камеры не сохраняются приложением и не отправляются разработчику.

Аналитика, реклама и трекинг

Приложение не использует рекламные SDK, аналитические SDK и сторонние инструменты трекинга. Данные пользователей не продаются и не передаются третьим лицам разработчиком.

Удаление данных

Пользователь может удалить локальные данные приложения, удалив приложение с устройства или очистив его данные в настройках системы.

Контакты

По вопросам конфиденциальности можно обратиться через GitHub Issues: https://github.com/Sorix/registrator-privacy/issues.
